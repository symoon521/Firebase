# 심, 밭다 (心, 밭다)

농업과 기술을 연결하는 혁신적인 플랫폼입니다. 농장주와 도시 소비자들이 직접 연결되어 신선한 농산물을 공유할 수 있는 서비스를 제공합니다.

## 🌱 프로젝트 개요

심, 밭다는 다음과 같은 기능을 제공합니다:

- **농장 탐색**: 다양한 지역의 농장을 검색하고 둘러볼 수 있습니다
- **작물 재배**: 원하는 농장에서 직접 작물을 재배할 수 있습니다
- **실시간 모니터링**: 스마트팜의 경우 실시간 스트림으로 작물 성장을 관찰할 수 있습니다
- **성장 기록**: AI 기반 성장 로그로 작물의 성장 과정을 추적합니다
- **AI 재배 팁**: 작물별 맞춤 재배 팁을 AI로부터 받을 수 있습니다

## 🛠 기술 스택

- **Frontend**: Next.js 15, React 18
- **언어**: JavaScript (TypeScript에서 변환됨)
- **스타일링**: Tailwind CSS, Shadcn/ui
- **AI**: Google AI (Gemini 2.0 Flash)
- **인증**: Firebase Auth (현재 목업 데이터 사용)
- **상태 관리**: React Context API
- **Form**: React Hook Form + Zod

## 🚀 시작하기

### 필요 조건

- Node.js 18 이상
- npm 또는 yarn

### 설치 및 실행

1. 의존성 설치
```bash
npm install
```

2. 개발 서버 실행
```bash
npm run dev
```

3. 브라우저에서 `http://localhost:9002` 접속

### 빌드

```bash
npm run build
npm start
```

## 📁 프로젝트 구조

```

│   ├── ui/              # UI 기본 컴포넌트
│   └── ...

├── hooks/               # 커스텀 훅
├── lib/                 # 유틸리티 및 설정

```

## 🎯 주요 기능

### 1. 농장 탐색
- 지역별, 작물별 농장 검색
- 스마트팜과 일반 농장 구분
- 실시간 스트리밍 지원

### 2. 작물 재배
- 농장별 재배 가능한 작물 확인
- 재배 시작 및 관리
- 성장 과정 추적

### 3. 대시보드
- 내가 재배 중인 작물 현황
- 실시간 성장 로그
- AI 재배 팁 제공

### 4. 사용자 관리
- 일반 사용자 / 농장주 구분
- 농장 등록 (농장주만 가능)
- 목업 인증 시스템
